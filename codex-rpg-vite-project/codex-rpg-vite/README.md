# Codex RPG

This is the Vite/React version of the Codex RPG app.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The finished static site appears in the `dist` folder.

## Deploy

For Netlify: upload this whole project or connect the GitHub repo. Netlify will run `npm run build`.

For GitHub Pages: build locally with `npm run build`, then upload the contents of `dist` to your Pages branch/root, or use a GitHub Pages workflow.
